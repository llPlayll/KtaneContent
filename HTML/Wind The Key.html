Keep Talking and Nobody Explodes Mod Wind-Up Key

On the Subject of Wind The Key

Morse meets Button meets Key. A Korsutton.

⚠️ IMPORTANT ⚠️
This is a module in the Wind-Up series.
Refer to Appendix WIND-UP KEY for important notes.

The display will show a number between 0 and 999.
Follow these rules in the order they are listed. Perform the first action that applies:

  1. If the display is double digits, hold the key and refer to "Releasing a Held Key".
  2. If there are more than 2 batteries on the bomb and there is a lit indicator with label CLR, turn and immediately release the key.
  3. If there is at least 1 port and there are more batteries than ports, refer to "Holding a Timed Key".
  4. If all displayed digits add up to at least 20 and there are more lit indicators than unlit, turn and immediately release the key
  5. If the serial number contains any digit from the module’s display, hold the key and refer to "Releasing a Held Key".
  6. If the number is prime, turn and immediately release the key
  7. If none of the above apply, refer to "Holding a Timed Key".

RELEASING A HELD KEY
The lower right light will start to flash a character in morse code. Based on the flashed character, you must release the key at a specific point in time:

  • A to H: release when the countdown timer has a 2 in any position.
  • I to P: release when the countdown timer has a 7 in any position.
  • Q to Z: release when the countdown timer has a 9 in any position.
  • Number: release when the countdown timer has the flashed number in any position.

HOLDING A TIMED KEY
Take the sum of both batteries and indicators.
You must hold the key for this specific number of seconds according to the timer. No more, no less.
Ignore the flashing light.

How to Interpret Morse

1. A short flash represents a dot.
2. A long flash represents a dash.
3. There is a long gap between letters.
4. There is a very long gap before the word repeats.
