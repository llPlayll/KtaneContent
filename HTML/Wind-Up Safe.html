This is a module in the Wind-Up series.
Refer to Appendix WIND-UP KEY for important notes.

The module contains a large, 12-sectioned combination safe with a keyhole in the center. Using the key, input a 3 part combination to disarm the module.

RULES:

  • The dial ticks count from 0-11, going clockwise.
  • The red tick on the dial is always 0.
  • The order of submission is Left, Right, Left.
  • To submit, turn to the first number, then second, then remove the key while the dial is set to the third.
    • You must turn the dial every step, even if the first step is 0 or the current step is the same as the previous.
  • The red LED will turn on while the combination is being tracked. The green LED will turn on during the third rotation.
  • If the green LED is off and the key is removed, the module will reset without striking.
  • Inputting more than 3 numbers or going clockwise first will stop tracking your combination and turn the LEDs off.

NOTE: once solved, the module will turn the green LED on when the key is inserted, and will not strike when removed.

SOLVE METHODS

The code on the tape will be modified multiple times depending on what ports are located on the bomb.

The conversions must go down the list in order and only done if the TYPE of port is on the module. If there are no ports, use all 6 ciphers.

RJ-45 // GRID
y x 0 1 2
0 7 2 4
1 3 5 11
2 10 6 8
3 1 9 0
  • IMPORTANT: any calculations past 0/11 will loop
    D1-D3 refers to the 3 digits in the code
  • DVI-D - XOR
    • Convert each digit to binary, adding 0s leftside if shorter than 4 bits:
    • XOR D1 & D2
    • XOR D2 & D3
    • XOR D3 & D1
    • Convert back to decimal
    • EXAMPLE: 9-6-5 --> 3-3-1
  • Parallel - Shift
    • Add the number of batteries to each digit (Use 6 if none)
    • EXAMPLE: BATTERY[6] [9-6-5] --> 3-0-11
  • PS/2 - Wheel
    • D1 /=/ D1 + D2 - D3
    • D2 /=/ D2 + D3 - D1
    • D3 /=/ D3 + D1 - D2
    • EXAMPLE: 9-6-5 --> 10-2-8
  • RJ-45 - Grid
    • D1 /=/ X = D2 MOD 3 // Y = D3 MOD 4
    • D2 /=/ X = D3 MOD 3 // Y = D1 MOD 4
    • D3 /=/ X = D1 MOD 3 // Y = D2 MOD 4
    • EXAMPLE: 9-2-5 --> 11-11-10
  • Serial - M-Shift
    • Shift each digit by the number of ports, starting backwards (subtract) (Use 9 if none)
    • Calculations are constrained to 4-number chunks, based on the digit
    • EXAMPLE: [2] would be 2-1-0-11
    • If you reach the edge of the chunk, reverse the direction
    • EXAMPLE: PORTS[9] [2] --> 1-0-11-0-1-2-1-0-11 --> 11
  • Stereo RCA - Doubles
    • Grab the 3rd and 6th digits in the serial number. These will be M1 and M2.
    • D1 = (D2 + M1) - (D3 + M2)
    • D2 = (D3 + M1) - (D1 + M2)
    • D3 = (D1 + M1) - (D2 + M2)
    • EXAMPLE: 9-7-6 M1[4] M2[6] --> 11-7-0
